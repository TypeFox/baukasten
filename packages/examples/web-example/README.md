# @baukasten/examples

Example implementations showcasing the @baukasten/ui component library.

## Getting Started

1. Install dependencies from the root:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run example
   ```

   Or from this directory:
   ```bash
   npm run dev
   ```

3. Open your browser to `http://localhost:3000`

## What's Included

This example application demonstrates:

- **Theme Switching**: Toggle between dark and light themes
- **Button Components**: Various button variants, sizes, and states
- **Input Fields**: Text inputs with labels, validation, and error states
- **Badges**: Status indicators with different variants and sizes
- **Form Example**: A complete registration form showcasing component composition

## Building for Production

```bash
npm run build
```

The built files will be in the `dist` directory.
